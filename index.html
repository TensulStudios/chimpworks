About

CHIMPWORKS is a multiplayer physics sandbox game, inspired by games like BONEWORKS and BONELAB, made by Temporary Games.

Server Creation

Server creation in CHIMPWORKS is not as easy as other games. CHIMPWORKS uses a token system so the game can stay free and open always.

You can learn how to make servers here.

Game is not currently released.

There is currently no release date for CHIMPWORKS.
